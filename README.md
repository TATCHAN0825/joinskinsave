# JoinSaveSkin

## Requirement
* gd

### Enable GD Library
`extension=php_gd.dll`のコメントアウトを解除します
```diff
- ;extension=php_gd.dll
+ extension=php_gd.dll
```

## Usage
* プラグインを入れてサーバーに参加する
* プレイヤーのスキンデータ(png)が保存されます！

`plugin_data\joinskinsave\プレイヤー名.png`
